# Dotfiles

Clean, colorful terminal setup for my Raspberry Pi environment.

## Layout

- `bash/.bashrc`
- `bash/.bash_aliases`
- `shell/.profile`
- `git/.gitconfig`
- `git/.gitignore_global`
- `install.sh`

## Quick Start

```bash
git clone git@github.com:erkcet/dotfiles.git
cd dotfiles
./install.sh
```

## Notes

- Existing local config files are backed up with a timestamp before linking.
- This repo is intentionally small and focused on daily shell + git workflow.
